# CompassTO

CompassTO provides two mobile clients for navigating Toronto social services:

- `apps/android`: React Native Android app
- `apps/ios`: native Swift + SwiftUI iOS app

Both clients include:

- social-services search and category filters;
- an eviction-prevention questionnaire and action plan;
- a senior and caregiver support area;
- official links and click-to-call actions;
- 12 languages: English, French, Portuguese, Spanish, Ukrainian, Russian, Arabic, Korean, Japanese, Simplified Chinese, Vietnamese and Tagalog;
- right-to-left layout for Arabic;
- on-device favorites and no storage of sensitive documents.

Each app has its own setup instructions in its local README.
